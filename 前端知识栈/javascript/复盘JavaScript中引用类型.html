Date вҖӢ

ECMAScriptзҡ„DateеҸӮиҖғдәҶJavaж—©жңҹзүҲжң¬дёӯзҡ„java.util.DateгҖӮдёәжӯӨпјҢDateзұ»еһӢе°Ҷж—ҘжңҹдҝқеӯҳдёәиҮӘеҚҸи°ғж—¶й—ҙж—¶(UTC)ж—¶й—ҙ1970е№ҙ1жңҲ1ж—ҘеҚҲеӨңпјҲйӣ¶ж—¶пјүиҮід»ҠжүҖз»ҸиҝҮзҡ„жҜ«з§’ж•°гҖӮ

иҰҒеҲӣе»әж—ҘжңҹеҜ№иұЎпјҢе°ұжҳҜз”Ёnewж“ҚдҪңз¬ҰжқҘи°ғз”ЁDateжһ„йҖ еҮҪж•°
