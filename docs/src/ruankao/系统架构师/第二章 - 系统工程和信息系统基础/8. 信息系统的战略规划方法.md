# 8. 信息系统的战略规划方法

每个企业都有自己的战略目标。开展多项业务需要数字化的支撑。

## 三个阶段

### 第一阶段 **以数据处理为核心** 围绕职能部门的需求（几种方法论可以结合使用）

- CSF 关键成功因素法

  关键信息集合 -> 确定开发优先次序

  **能抓住主要矛盾**

- SST 战略集合转化法

  （使命、目标、战略） -> 信息系统的战略目标

- BSP 企业系统规划法

  自上而下规划，自下而上实现

### 第二阶段 **以企业内部 MIS【管理系统】为核心** 围绕企业整体需求

- SDP 战略数据规划法： 主题数据库
- IE 信息工程法
- SG 战略栅格法

### 第三阶段 综合考虑企业内外环境 **以集成为核心**，围绕企业战略需求

- 价值链分析法
- 战略一致性模型
